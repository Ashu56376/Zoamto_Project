🍽️ Zomato Data Analysis
📌 Project Overview

This project focuses on analyzing Zomato restaurant data to extract meaningful insights about restaurant trends, customer preferences, ratings, pricing, and location-based performance.
The analysis helps in understanding market patterns and business opportunities in the food industry.

🎯 Objectives

Analyze restaurant ratings distribution

Identify top locations with highest restaurant density

Compare online delivery vs dine-in trends

Analyze price range distribution

Find most popular cuisines

Detect factors affecting restaurant ratings

🛠️ Tools & Technologies Used
